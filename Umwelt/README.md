# UMWELT · Braitenberg 沙盒

一个基于 Braitenberg 车辆理论的交互式神经行为模拟器。通过调整传感器与电机之间的神经连线，观察一只小虫如何在充满食物与威胁的世界中生存。

## 核心概念

[Braitenberg 车辆](https://en.wikipedia.org/wiki/Braitenberg_vehicle) 是一种思想实验：仅凭传感器到电机的直接连线，就能涌现出趋近、回避、追击、依恋等复杂行为。UMWELT 将这个实验做成可交互的沙盒。

**四种基础行为（由连线方式决定）：**

| 行为 | 连线模式 | 现象 |
|------|----------|------|
| 趋近 | 同侧兴奋 | 靠近食物，速度随信号增强 |
| 追击 | 交叉兴奋 | 持续转向食物方向 |
| 依恋 | 同侧抑制 | 在食物旁边减速停留 |
| 回避 | 交叉抑制 | 感知到食物时转向远离 |

除了四种基础行为，还可以插入**中间神经元**和**调制神经元**来构建更复杂的网络——例如振荡器、记忆回路、饥饿驱动的行为切换等。

## 界面说明

### 顶栏
- **存活 / 食物 / 代数**：当前轮次的基本统计
- **暂停 / 速度**：1/8x 到 2x 倍速，方便观察细节或快进
- **环境**：调整食物与威胁的数量（0 到最大值）
- **重新投入**：重置小虫，代数 +1
- **重置连线**：将神经图恢复为默认状态

### 传感器选择
小虫身上有 12 个方向传感器，分布在一个圆环上：
- **F0–F5**：食物传感器，按方向排列（正前、右前、右后、正后、左后、左前）
- **T0–T5**：威胁传感器，同样的 6 个方向

点击节点可启用或关闭对应传感器。传感器越多，感知越强，但能量消耗也越大。

每个传感器支持两种模式（右键切换）：
- **绝对值**：当前方向的刺激强度
- **差分**：信号变化速率（对运动中的目标更敏感）

### 神经连线图（编辑器）
点击「编辑回路」进入暂停编辑模式：
- **从节点拖出**：创建新连线，释放时弹出菜单选择兴奋或抑制
- **右键连线或节点**：删除
- **双击空白处**：插入中间节点（可用于构建更复杂的网络）

**身体参数面板**：编辑模式下可调节两个形态参数：
- **转向缩放**（0.1–10.0）：放大或缩小转向力度
- **速度缩放**（0.1–5.0）：放大或缩小前进速度

### 底部状态栏
实时显示各传感器输出、左右腿驱动强度、速度与能量条，以及行为事件日志。

## 感知系统

小虫通过**化学场扩散**感知世界，而非简单的距离检测。食物和威胁各自向周围散发浓度场，传感器在锥形感知区域内采样场的浓度值。

| 参数 | 食物场 | 威胁场 |
|------|--------|--------|
| 扩散系数 | 8.0 | 6.0 |
| 衰减率 | 0.55 | 0.50 |
| 发射强度 | 1.2 | 1.5 |
| 感知半径 | 300px | 250px |

### 本体感受器（始终激活）
- **P_speed**：当前速度的归一化值
- **P_turn**：当前转速的归一化值
- **P_hunger**：饥饿程度（`1 - 当前能量 / 最大能量`），可用于构建饥饿驱动行为

## 神经网络

### 节点类型
- **传感器节点**：接收外界或本体信号
- **电机节点**：左腿 / 右腿，驱动运动
- **中间神经元**：双击空白处创建，可用于信号中继、延迟、振荡
- **调制神经元**：对其他神经元的输出进行乘性缩放，用于实现门控和行为切换

### 高级动力学
抑制性连接支持**反跳振荡**（Post-Inhibitory Rebound），通过 `tau_charge` / `tau_discharge` 参数控制充放电时间常数，可实现类 Matsuoka 振荡器等节律行为。

## 能量系统

小虫有 100 点能量，耗尽即死亡：
- 每秒基础衰减 1.5
- 运动消耗：平均电机输出 × 2.0 / 秒
- 传感器消耗：激活传感器的信号强度之和 × 0.8 / 秒
- 吃到食物：+25（食物会在地图其他位置重生）
- 威胁伤害：进入威胁区域后，按浓度持续受损（伤害率 = 30 × (浓度 - 0.3) / 秒）
- 基础驱动：电机始终有 0.48 的基线输出（tonic drive），即使没有任何传感器输入

## 项目结构

```
src/
├── main.js          # App 主类，游戏循环
├── config.js        # 全局常量、传感器/连线定义、行为预设
├── world.js         # 世界状态、化学场扩散、物理步进、能量逻辑
├── neural.js        # NeuralGraph：节点/边管理、信号求值、行为推断
├── math.js          # 工具函数（clamp、lerp、角度包裹等）
├── style.css        # 全局样式，支持亮/暗主题
├── renderer/
│   ├── world.js     # Canvas 世界渲染（小虫、食物、威胁、轨迹）
│   ├── graph.js     # 神经图 Canvas 渲染与交互（拖拽、右键菜单）
│   └── sensor-map.js # 传感器环形图渲染
└── ui/
    ├── topbar.js    # 顶栏 DOM 操作
    ├── footer.js    # 底部数据栏与事件日志
    ├── sidebar.js   # 侧边栏容器，整合传感器图和神经编辑器
    ├── editor.js    # 神经编辑器面板（身体参数、编辑/运行按钮）
    └── death.js     # 死亡诊断弹窗
```

## 技术栈

- **Vite 7**：构建工具，无其他运行时依赖
- 纯原生 JS + Canvas 2D API
- 固定时间步长 60Hz 物理模拟，与渲染帧解耦
- 化学场基于离散扩散方程（近似 Bessel K₀ 分布）
